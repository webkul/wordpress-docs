Feature Catalog

Every shipped feature, grouped by surface. Click through for the full reference page.

Auction types

TypeWhat it does
Standard (Forward)Highest-bidder-wins. The classic format.
ReverseLowest bid wins. For procurement / contractor markets.
Sealed / BlindBids hidden until the auction closes.
Proxy / Auto-BidUser sets a max; system bids on their behalf in increments.
SilentAll-pay style with bid amounts shown publicly only at close.
CharityIssues tax-deductible receipts; donor wall.
PennyPay-per-bid model with bid packs.
DutchDescending price; first to accept wins.
SeatedLive auctioneer-led with paddle/seat assignments.
Unique-BidLowest unique bid wins.

Bidding engine

  • Real-time AJAX live bidding with adaptive polling (5–10 s normal → 2–3 s last 2 min)
  • Soft-close anti-sniping with configurable window, extension, max-extension cap
  • Server-locked bid placement — two simultaneous bidders can never both win the same price
  • Hash-chain bid audit ledger — tamper-evident
  • Bid queue for high-traffic auctions
  • Proxy / auto-bidding
  • Bid retraction window
  • Bid cooldown, max bids per user, per-auction caps
  • Quick-bid buttons, one-click rebid
  • Anonymous bidding, masked names
  • Absentee / pre-bidding before auction starts

Pricing

  • Fixed bid increment
  • Variable / tiered increment table
  • Percentage-based increment
  • CSV import / export of increment rules
  • Reserve price + indicator
  • Buy It Now with auto-disable rules

Customer experience

  • My Auctions — active bids, won, lost, watchlist tabs
  • My Wallet — top up, transfer, ledger
  • Withdrawal — bank / PayPal / custom payout methods
  • Watchlist, favorites, saved searches
  • Auction comparison (up to 3 lots side-by-side)
  • Shipping estimator
  • Contact-seller messaging
  • Social sharing
  • Print auction details

Wallet & payments

  • Wallet system — top-up, transfer, ledger, OTP
  • Auto-charge winner on auction close
  • Stripe Connect for marketplace payouts
  • Monetization — listing fees, buyer's premium, late penalty, featured boost, bid fee

Multi-vendor

Native compatibility with:

  • WCFM
  • Dokan
  • WC Vendors
  • YITH Multi-Vendor

The plugin defers vendor onboarding, dashboards, and payouts to your active multi-vendor plugin and surfaces auction-specific UI inside it.

Premium features

  • Smart features — hot/trending detection, recommendations, auction-of-the-day
  • Psychology triggers — watchers count, urgency, scarcity, masked-name engagement
  • Gamification — points, badges, leaderboards, achievements
  • Fraud detection — bid rate limit, IP rules, email verify, SSL gate
  • Compliance — KYC, age gate, terms acceptance, donor wall, tax receipts, GDPR
  • Social sharing — Facebook, Twitter, WhatsApp, Pinterest, LinkedIn
  • Email digest, auto-relist

Notifications

21 transactional emails, all editable from WooCommerce → Settings → Emails:

Bid placed, outbid, auction starting, started, ending soon, won, lost, payment reminder, payment confirmation, payment overdue, auction failed, auction relisted, watchlist activity, registration, new-auction-from-seller, lot approved, lot rejected, seller application received / approved / rejected.
