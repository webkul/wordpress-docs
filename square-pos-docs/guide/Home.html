WooCommerce POS Square Payment Terminal

Welcome to the documentation for WooCommerce POS Square Reader Native SDK Integration. This plugin enables WooCommerce store administrators to accept payments at their physical point-of-sale (POS) outlets using the Square payment terminal and SDK.

Introduction

The WooCommerce POS Square Reader Native SDK Integration plugin bridges the gap between your online WooCommerce store and physical checkout counters. By leveraging the Square Payment SDK, it provides a seamless, secure, and robust payment experience for both POS agents and customers.

PRO TIP

This is an add-on plugin. You must have the Point of Sale System for WooCommerce installed to use this integration.

Key Features

  • Swift Card Payments: Accept Visa, Mastercard, American Express, and Discover directly at your POS terminal.
  • SDK Integration: Utilizes the native Square Payment SDK for high reliability and security.
  • Default Payment Method: Set Square as the default payment option for faster checkouts.
  • Test Mode Support: Use Square's simulator and sandbox tokens for safe testing before going live.
  • Real-time Sync: Automatically sync payment status between the POS and WooCommerce.
