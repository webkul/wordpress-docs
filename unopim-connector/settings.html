Settings Overview

Where to find every setting in the connector. Use this page as the index — every row links to the detail page.

The six admin pages

The connector adds a single top-level menu, UnoPim Configuration, with six sub-pages. They're available to administrators and shop managers.

PageWhat it doesDetail page
UnoPim ConfigurationYour UnoPim address, sign-in details, and the Test & Save buttonConnection
Channels & StoresChannel-to-site mapping, per-channel pull/push toggles, sync intervalChannels & Stores
Field MappingThree sub-tabs: Core Fields, Attributes, CategoriesField Mapping
SyncManual triggers — full pull, single-entity pull, push everythingSync
Scheduled JobsRecurring sync jobs (per channel, per language, per action)Scheduled Jobs
LogsA filterable record of every sync — what synced, what failed, what was skippedLogs

A separate General Settings form lives alongside the Connection page — see General Settings.

What each page controls

Connection

  • Your UnoPim address and sign-in details
  • Test & Save verifies the credentials before storing them
  • Connection status badge

Channels & Stores

  • Channel-to-site mapping
  • Per-channel Pull enabled, Push enabled, Sync interval, Cooldown seconds
  • Per-channel languages and currencies
  • Refresh channels button

Field Mapping → Core Fields

  • The pinned rows: SKU, Name, Regular Price, Sale Price, Stock Quantity, Stock Status, Description, Short Description, Publish Status
  • The default product family input (used as a push fallback)
  • Apply WC defaults and Save mapping

Field Mapping → Attributes

  • Per (channel, language) custom-attribute mapping
  • Live UnoPim attribute search
  • + Add WooCommerce field, + Add custom column, Auto-map, Save preset, Load preset

Field Mapping → Categories

  • Per-channel WooCommerce category ↔ UnoPim category mapping

Sync

  • Pull Full — kicks off a full UnoPim → WooCommerce sync
  • Per-entity pull buttons — pull just attributes, just categories, just products, etc.
  • Push All Products / Categories / Attributes
  • Channel and language scope picker

Scheduled Jobs

  • A list of your recurring sync jobs
  • Create / edit form: name, action, scope, schedule
  • Pause, resume or delete each job

Logs

  • A paginated table of every sync operation
  • Filters by status, scope, entity, time window
  • A flush button to trim old rows

General Settings

  • Dry-run toggle
  • Writeback delete toggle
  • On-delete behaviour
  • Remove data on uninstall

See General Settings for the field-by-field reference.
