Installation

To install the WooCommerce Shopify Connector, follow these steps:

  1. Access WordPress Admin: Login to your WordPress Admin Panel.

  2. Navigate to Plugins: In the Dashboard, hover over the Plugins menu and select Add New.

  3. Upload Plugin: Click the Upload Plugin button at the top of the page.

  4. Select File: Click Choose File and browse for the plugin zip file.

  5. Install Now: Click the Install Now button.

  6. Activate Plugin: Once installed, click Activate Plugin to complete the process.

Upon successful activation, the module is ready for use.
