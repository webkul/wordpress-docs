Features of WooCommerce Shopify Connector

The WooCommerce Shopify Connector provides a complete solution to synchronize data between WooCommerce and Shopify stores efficiently.

Key Features

  • Map Shopify collections with WooCommerce categories.
  • Import Shopify products into WooCommerce with ease.
  • Set a default category for imported products.
  • Support for Simple and Variable product types (import/export).
  • Import complete product details:
    • Images (base & gallery)
    • Name, price, SKU
    • Description, quantity, weight
    • Attributes
  • Imported products function like native WooCommerce products.
  • One-click manual product update option.
  • Export WooCommerce products to Shopify.
  • Import Shopify orders into WooCommerce.
  • Set default order status for imported orders.
  • Add and manage multiple Shopify accounts.
  • Export orders to Shopify with a single click.
  • Real-time auto-sync for orders and products.
  • Auto-sync product details like price, SKU, stock, and category.
  • Apply price rules for import/export with real-time updates.
  • Two-way real-time product synchronization (WooCommerce ↔ Shopify).
  • Two-way automatic inventory management on new orders.
  • Sync unlimited orders without any monthly fees.
  • Import additional data:
    • Customers
    • Pages
    • Blogs
    • Coupons
  • Import and manage shipping and tax from Shopify.
  • Display Shopify tracking details in WooCommerce orders.

Advanced Features

  • Real-Time Sync – Fields to Update
    Control which product fields (price, images, categories, SKU, stock, etc.) are synchronized between WooCommerce and Shopify in real time.

Important Notes

  • For sale products in Shopify:
    • Price = Sale price
    • Compare at Price = Regular price
  • Only products with up to 3 variation attributes can be exported to Shopify.
  • Shopify store location must be configured for product export.
  • Fully compatible with GraphQL API.

Real-Time Synchronization

The connector supports real-time synchronization in both directions (Shopify ↔ WooCommerce):

  • Automatic Creation
    New products created in Shopify are automatically created in WooCommerce.

  • Automatic Update
    Product changes (price, stock, SKU, description, etc.) are synced instantly.

  • Automatic Deletion
    Deleted products in Shopify are also removed from WooCommerce.

Automatic Inventory Management

  • Stock levels are automatically updated when products are sold or modified.
  • Out-of-stock products in Shopify are reflected in WooCommerce.
  • Works in both directions based on sync configuration.
  • Reduces manual inventory management efforts.
