WooCommerce Point of Sale (POS)

WooCommerce Point of Sale (POS) extension allows store owners to seamlessly manage online WooCommerce stores alongside physical retail outlets using a unified POS system.

The system provides a dedicated POS front-end interface for physical stores, where store agents can manage sales, customers, and inventory efficiently.

Overview

  • Manage online and offline stores together
  • Assign store agents to specific outlets
  • Supports online and offline sales with automatic synchronization
  • Optimized for fast in-store checkout
  • Built with React for a modern, high-performance UI

Offline data (sales & customers) automatically syncs with the WooCommerce store once the system goes online.

Important Notes

  • WooCommerce POS supports Simple and Variable products only
  • Backorders are not supported in POS Backorders are avoided to maintain speed, simplicity, and a smooth in-store customer experience

POS Applications

  • Web-based POS Panel
  • Desktop POS App (PWA)
  • Tablet & Mobile PWA App
  • Compatible with browsers on desktops, laptops, tablets, and iPads

Core Features

Store & Inventory Management

  • Create multiple store outlets
  • Assign multiple sales agents per outlet
  • Centralized or Master Stock inventory
  • Bulk master stock assignment
  • Real-time inventory synchronization
  • Visibility of out-of-stock products

Sales & Checkout

  • Barcode scanning (SKU or Product ID)
  • Support for Code-128 barcodes
  • Multiple payment methods & split payments
  • Coupons & cart-level discounts
  • Inline price editing
  • Hold cart for later checkout
  • Dark mode support
  • Unit price selling (weight-based products)

POS Operations

  • Offline order creation & sync
  • Drawer management (open/close, reports)
  • Order receipt via Email & WhatsApp
  • Auto-print invoice support
  • Keyboard shortcuts
  • Custom order prefix
  • POS product pinning

Advanced Integrations

  • WhatsApp Notifications (via Twilio)
  • Broadcast & personalized WhatsApp messages
  • Node.js server integration
  • WooCommerce HPOS compatibility
  • Multisite support
  • Desktop & Mobile PWA support

Installation

  1. Upload the POS plugin ZIP file via WordPress Admin → Plugins → Add New → Upload Plugin
  2. Click Install Now
  3. Activate the plugin after installation
  4. Flush permalinks via Settings → Permalinks

Admin Configuration

Navigate to: WooCommerce Point of Sale → Settings

Configuration Sections

  • General Settings
  • Customer Settings
  • POS Mass Assign
  • Barcode Settings
  • Web App Settings
  • Media Settings
  • Payment Methods
  • Invoice Printer
  • Appearance
  • Endpoints
  • Custom CSS & JS
  • WhatsApp API Configuration
  • Node Server Configuration

Inventory Types

  • Master Stock – Central stock shared across outlets
  • Centralized Inventory – Unified stock system

POS Wizard Setup

The POS Wizard allows instant setup:

  1. Configure API credentials
  2. Select inventory type
  3. Configure web app branding
  4. Set default customer
  5. Create or import outlets

Outlet Management

  • Create & import outlets via CSV
  • Assign products & stock per outlet
  • Enable all products for an outlet
  • Activate / Deactivate outlets
  • View outlet-wise reports

POS Users

  • Create & manage POS users
  • Assign multiple users to one outlet
  • Set default POS customer
  • Bulk delete POS users

Product & Barcode Management

  • Generate barcodes using:
    • Product ID
    • SKU
  • Supports variable products
  • Print barcodes in bulk
  • Search products using:
    • SKU
    • Product ID
    • GTIN / UPC / EAN / ISBN

Orders & Drawer Management

  • View POS & website orders
  • Track refunds
  • Daily drawer open/close reports
  • Email drawer reports to admin
  • Sale history tracking

Reports

Admin Reports

  • Orders
  • Customers
  • Stock (Low / Out of Stock / High Stock)
  • Export data (Orders, Coupons, Taxes)

POS Agent Reports

  • Gross & Net Revenue
  • Orders & Average Order Value
  • Refunds & Discounts
  • Payment-wise breakdown
  • Date range & product filters
  • Area, Line & Bar charts

Invoice Templates

  • Create multiple invoice templates
  • Visual invoice editor
  • Customize branding & layout
  • Assign printer types (A3, A4, A5, Thermal, etc.)

WhatsApp Greetings

  • Create & manage greeting templates
  • Auto or manual order receipt delivery
  • Bulk messaging support
  • Template enable/disable controls

POS Functionalities (Agent Side)

  • Product search & categories
  • Custom products (offline-only items)
  • Customer assignment & management
  • Cashier drawer handling
  • Hold & resume carts
  • Offline sales sync
  • Inline price edits
  • Unit-price selling
  • Keyboard shortcuts

Desktop & Mobile POS

Desktop App

  • Install directly from Chrome
  • Faster performance
  • Native-like experience

Mobile PWA

  • Works on low internet
  • Add to home screen
  • Optimized for portrait view

System Requirements

  • Minimum resolution: 1280×800
  • Screen size: 7.5 inches or larger
  • Works on:
    • Desktop PCs
    • Laptops
    • Tablets
    • iPads
    • Macs

Additional Notes

  • Offline mode works only with Master Stock
  • Shipping visible only if configured
  • Development version available at additional cost
  • Mobile native app available separately

Conclusion

WooCommerce Point of Sale is a complete retail POS solution that bridges online WooCommerce stores with physical outlets, providing fast checkout, offline selling, centralized inventory, and powerful reporting — all in one system.
