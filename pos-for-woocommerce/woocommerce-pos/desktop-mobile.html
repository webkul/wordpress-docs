WooCommerce POS - Desktop & Mobile Applications Guide

PWA Desktop App Setup

Overview

The WooCommerce POS system offers a Progressive Web App (PWA) that can be installed as a desktop application, providing a native app-like experience with enhanced performance.

Installing Desktop Application

Installation Steps:

  1. Open your WooCommerce POS in Google Chrome Browser
  2. Look for the "+" (plus) sign in the address bar or browser menu
  3. Click on the "+" sign to initiate installation
  4. Confirm installation when prompted
  5. Desktop app window: New desktop app window appears after installation

Alternative Installation Method:

  • Click on the three-dot menu in Chrome
  • Select "Install [App Name]" or "Create Shortcut"
  • Check "Open as window" option
  • Click "Install" or "Create"

Desktop App Benefits

Performance Advantages:

  • Faster Management: Salesperson can manage POS products, orders, transactions faster from desktop app
  • Dedicated window without browser navigation
  • Optimized for desktop use
  • Quick launch from desktop or taskbar

Operating System Compatibility:

  • Operating system compatibility: Install desktop application for any operating system from Google Chrome Browser
  • Windows
  • macOS
  • Linux
  • Chrome OS

Desktop App Features

Functionality:

  • Full POS capabilities
  • Works online and offline
  • Standalone application window
  • Native-like experience
  • System tray integration (on supported OS)
  • Automatic updates

Mobile POS Installation

PWA Mobile Application Overview

Mobile PWA Features:

  • PWA POS mobile application: Works in low internet connection
  • Loads on home screen like native app
  • No app store download required
  • Instant updates
  • Cross-platform compatibility

Installing on Mobile Devices

Installation Steps:

  1. Open POS Link: Open POS link in browser of mobile device (Chrome, Safari, or Firefox)
  2. Access Menu: Click on menu icon of browser (three dots or share icon)
  3. Add to Home Screen: Select "Add to Home Screen" option
  4. Confirm Installation: Follow on-screen prompts to complete installation

Platform-Specific Instructions:

For Android (Chrome):

  1. Open POS URL in Chrome
  2. Tap the three-dot menu
  3. Select "Add to Home screen"
  4. Name the app
  5. Tap "Add"

For iOS (Safari):

  1. Open POS URL in Safari
  2. Tap the Share button (box with arrow)
  3. Scroll and tap "Add to Home Screen"
  4. Name the app
  5. Tap "Add"

After Installation

App Display:

  • App icon: Visible on home screen of mobile after installation
  • Custom icon with POS branding
  • Looks like a native app

Launching the App:

  • Launch app: Click on application icon to launch it
  • Opens in full-screen mode
  • No browser UI visible
  • Native app experience

Mobile Access Features

Mobile Capabilities:

  • Mobile access: Users access POS products, categories, orders through mobile devices in portrait view
  • Touch-optimized interface
  • Swipe gestures supported
  • Mobile-friendly layouts

Mobile View Options:

  • Portrait mode (default)
  • Landscape mode (for tablets)
  • Responsive design
  • Adaptive UI elements

Working in Low Connectivity

Offline Functionality

Low Connection Performance:

  • PWA mobile app: Works in low internet connection
  • Continues functioning without internet
  • Stores data locally
  • Processes transactions offline

Offline Capabilities:

  • Offline functionality: Point Of Sale systems work both online and offline with synchronization feature
  • Complete sales without internet
  • Access product catalog
  • Manage customers
  • Process payments
  • Hold and complete orders

Offline Data Management

Data Synchronization:

  • Offline data sync: All sales and customer data of offline store sync with online store whenever going online
  • Automatic background sync
  • Queue-based sync system
  • No data loss

Viewing Offline Orders

Offline Sales Section:

  • Offline orders: Sales orders processed in offline mode visible under POS > Order > Offline Sales
  • Track unsynced orders
  • Monitor sync status
  • Review offline transactions

Synchronization Options

Manual Sync:

  • Sync functionality: By Sync Order button all offline orders synced to online store
  • Click "Sync Order" button
  • Manually trigger synchronization
  • Immediate data transfer

Automatic Sync:

  • Auto-sync: Enable option for offline orders to sync automatically when creating any order online
  • Configure in settings
  • Background synchronization
  • No manual intervention needed

Offline Best Practices

Preparation:

  1. Ensure products synced before going offline
  2. Verify customer data is current
  3. Check local storage capacity
  4. Test offline functionality regularly

During Offline Operation:

  1. Monitor storage space
  2. Keep track of offline orders
  3. Sync as soon as connection available
  4. Backup data regularly

Hardware Integration (Barcode Scanner, Printer)

Barcode Scanner Integration

Scanner Support:

  • Barcode readers: Can be used to add products to cart
  • Barcode scanning: POS users add product to cart by scanning barcode (Generated based on SKU or Product ID)

Barcode Format:

  • Barcode format: POS application supports Code-128 barcodes
  • Industry-standard format
  • High compatibility
  • Reliable scanning

Scanner Types Supported:

  • USB barcode scanners
  • Bluetooth barcode scanners
  • Built-in camera scanning (mobile)
  • Handheld scanners
  • Fixed-mount scanners

Scanner Setup:

  1. Connect scanner to device
  2. Ensure scanner is recognized by system
  3. Scanner acts as keyboard input
  4. Scan barcodes directly into POS
  5. No additional configuration needed (most cases)

Printer Integration

Thermal Printer Support:

  • Printers: Thermal printer support mentioned
  • Receipt printers
  • Label printers
  • Standard paper printers

Supported Printer Types:

  • Epson TM-T88V Thermal Printer
  • Star Micronics printers
  • USB receipt printers
  • Network printers
  • Bluetooth printers

Printer Configuration:

  1. Connect printer to device
  2. Install printer drivers (if needed)
  3. Select printer in POS settings
  4. Configure paper size (A3, A4, A5, A6, Thermal)
  5. Test print functionality

Performance Tips

Optimizing Speed:

  1. Use Modern Browsers: Chrome, Safari, or Firefox latest versions
  2. Clear Cache Regularly: Remove old cached data
  3. Stable Connection: Use reliable internet connection
  4. Close Unused Tabs: Free up browser resources
  5. Update Regularly: Keep POS and browser updated
  6. Hardware Acceleration: Enable in browser settings

Network Optimization:

  1. Use wired connection when possible
  2. Ensure adequate bandwidth
  3. Minimize network congestion
  4. Use local caching
  5. Enable offline mode for unstable connections

Device Maintenance:

  1. Keep operating system updated
  2. Remove unnecessary applications
  3. Maintain free storage space
  4. Restart device regularly
  5. Monitor system performance
