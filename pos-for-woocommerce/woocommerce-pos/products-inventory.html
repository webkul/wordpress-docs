WooCommerce POS - Products & Inventory Management Guide

Assigning Products to Outlets

Overview

Manage which products are available at each outlet location through the product assignment interface.

Accessing Product Assignment

Navigate to: WooCommerce Point Of Sale > Products

Manual Product Assignment

Step 1: Access Manage Products Tab

  • Click on Manage Products tab
  • View list of all available products

Step 2: Enable Products for Outlet

  • Admin can assign products to outlet by enabling products for outlet
  • Check the box next to each product to enable it
  • Select specific products based on outlet requirements

Step 3: Save Changes

  • Click Save to apply product assignments
  • Verify products appear in outlet

Bulk Product Assignment

Enable All Products:

  • For particular outlet by tapping Enable All Products button
  • Quick assignment for new outlets
  • Useful when all products should be available

POS Stock Assignment

Assigning Stock:

  1. Assign POS stock to products accordingly
  2. Admin edits outlet and manages products to add POS stock to product accordingly
  3. Set stock quantities per outlet

Product Status:

  • Default product status for outlet can be set enable or disable
  • Control product visibility per location
  • Manage seasonal or location-specific inventory

Stock Management (Master vs Outlet Stock)

Inventory Type Selection

Select Inventory Type:

Navigate to settings and choose between:

  • Master Stock: Centralized inventory management
  • Outlet Stock: Individual outlet inventory management

Master Stock Management

Overview:

Master Stock provides an effective sales and inventory management system for admin with centralized control.

Benefits:

  • Centralized inventory can be allowed by admin
  • Simplified stock management across all outlets
  • Bulk operations support

Master Stock Assignment:

For Simple Products:

  1. Navigate to Product tab
  2. Admin can apply master stock to all simple products at once
  3. Select all products using checkbox
  4. Click Edit button
  5. Apply master stock to selected products
  6. Save changes

Using Master Stock Tab:

  • Click on Master Stock tab
  • Enter master stock for products
  • Add product stock for store under "Stock Quantity"
  • Save master stock settings

Bulk Assignment:

  • Master stock assignment can be assigned by admin to all products at a time
  • Master stock assignment can be done in bulk
  • Select multiple products and apply stock simultaneously

Master Stock Management for Variable Products

Variable Product Configuration:

  1. Admin provides master stock to variable product by adding master stock to each variation
  2. Access product edit page
  3. Navigate to variations section

Setting Variation Stock:

  1. Admin will set variation stock by clicking Set Variation Stock button
  2. Add POS stock quantity to each variation accordingly
  3. Configure stock for each product variant

Individual Variation Management:

  • Set different stock levels for each variation
  • Manage size, color, or other attribute combinations
  • Track variation-specific inventory

Barcode Generation & Printing

Barcode Overview

Supported Formats:

  • POS application supports Code-128 barcodes
  • Compatible with standard barcode readers
  • Generate barcodes by Product ID or SKU

Accessing Barcode Generation

Navigate to: WooCommerce Point of Sale > Products

Generating Barcodes for Simple Products

Step 1: Select Generation Method

Choose between:

  • Generate barcode by ID: Based on product ID
  • Generate barcode by SKU: Based on product SKU

Step 2: Select Products

  • Select products for which to generate barcodes
  • Use bulk selection for multiple products

Step 3: Generate

  • Click generate button
  • View generated barcodes
  • Preview before printing

Generating Barcodes for Variable Products

Variable Product Barcode Process:

  1. Click Barcode button to view barcodes for all variations
  2. Select product from product list
  3. Select Generate Barcode by ID or Generate Barcode by SKU option

View Variation Barcodes:

  • Click barcode button to view barcode (ById or SKU) for each variation
  • Each variation gets unique barcode
  • Manage barcodes per variation

Printing Barcodes

Print Setup:

  1. Admin can easily print barcode for product based on product id or SKU
  2. Click Print Barcode button
  3. Add quantity of barcode to print
  4. Configure print settings

Print Settings:

  • Set number of copies
  • Select label size
  • Configure layout

Print Barcode for Variations:

  • Print barcode for each variation accordingly
  • Separate labels for each variant
  • Bulk printing support

Barcode Settings Configuration

Customize Barcode Appearance:

  • Background Color: Select background color of barcode
  • Width: Set barcode width
  • Height: Set barcode height
  • Display Title: Barcode title will display or not
  • Text Position: Text position of barcode (top, bottom, none)
  • Text Size: Set text size of barcode
  • Print Preview: Set print preview whether portrait or landscape

Using Barcodes in POS

Barcode Scanning:

  • POS users can add product to cart by scanning barcode
  • Barcode generated based on SKU or Product ID
  • Barcode available on product labels

Barcode Readers:

  • Barcode readers can be used to add products to cart
  • Compatible with USB and Bluetooth scanners
  • Quick product lookup

SKU Search:

  • POS users can search for products based on SKU
  • Manual SKU entry supported
  • Alternative to barcode scanning

Price & Discount Rules

Unit Price Feature

Weight-Based Pricing:

  • Admin or sales agent can set unit price based on weight of product
  • Enable Unit Price Feature in settings
  • Admin set this field as enable to allow selling product according to weight

How It Works:

  1. Sales agent enters product weight
  2. System automatically calculates price
  3. Price updates in real-time
  4. Modify weight as needed

Price Editing

Cart Price Modification:

  • POS user can edit price of items in cart
  • Override default pricing when authorized
  • Apply manual discounts

Customer-Based Discounts

Overview:

Customer-based discount helps enhance store sales by attracting customers through targeted pricing.

Setting Up Customer Discounts:

  1. Admin enters different prices for different customer groups
  2. Create customer segments
  3. Assign discount rules to groups

Discount Application:

  • Customer-based discount applicable for few products
  • Discount applied automatically at checkout
  • Based on customer group membership

When POS User Selects Customer:

  • POS user chooses particular customer group
  • Discount coupon applied to particular order automatically
  • Price adjusts based on customer tier

Coupon Management

Creating Coupons:

  1. Admin can create coupon for POS products
  2. Base coupons on different customer groups
  3. Add email addresses for targeted coupons
  4. Set expiration dates and usage limits

Applying Coupons:

  • At time of checkout, POS agent can apply coupon to cart
  • Enter coupon code manually
  • System validates coupon eligibility
  • Discount applies automatically

Coupon Types:

  • Percentage discounts
  • Fixed amount discounts
  • Free shipping
  • Buy X Get Y offers

Product Categories & Tags

Category UI Configuration

UI Style Selection:

  • Admin can select old or new UI style
  • Choose interface that suits workflow
  • Configure from appearance settings

Category Navigation in POS

Category Lists:

  • POS user may view category lists
  • Browse products by category
  • Select category to choose products

Category-Based Product Display:

  • POS user can look for products in respective categories
  • Organized product browsing
  • Quick product location

Search Functionality:

  • Using search bar and product category, POS User can search for any product of any product type
  • Combine search with category filtering
  • Fast product lookup

Supported Product Types

Compatible Product Types:

  • Works with Simple Products only
  • Works with Variable Products only
  • Full support for both types

Product Type Features:

  • Simple products: Single price point
  • Variable products: Multiple variations with different prices and stock

Inventory Alerts

Low Stock Warnings

Configuration:

  • Quantity For Low Stock Warning: Enter maximum quantity for products to show low stock warnings
  • Set threshold per product or globally
  • Receive alerts when stock falls below threshold

Low Stock Monitoring:

  • Low in Stock report: Admin can see list of all products low in stock
  • Regular monitoring recommended
  • Proactive restocking

Out of Stock Management

Out of Stock Visibility:

  • Visibility of product out-of-stock
  • Control whether out-of-stock products appear in POS
  • Configure in settings

Out of Stock Reports:

  • Out of stock list: All products out of stock will be visible
  • Identify products needing immediate restocking
  • Track out-of-stock duration

Stock Reports

Most Stock List:

  • All products with highest stock will be listed
  • Identify slow-moving inventory
  • Optimize stock levels

Stock Updates:

  • POS now updates product stock when order status updated
  • Real-time inventory synchronization
  • Accurate stock tracking

Bulk Product Operations

Bulk Master Stock Assignment

Mass Stock Updates:

  • Master stock assignment can be assigned by admin to all products at a time in bulk
  • Select multiple products
  • Apply stock quantity to all selected

Process:

  1. Navigate to Products page
  2. Select all products using checkbox
  3. Click Edit button
  4. Enter master stock quantity
  5. Apply to all selected products

Bulk Barcode Generation

Generate Multiple Barcodes:

  1. Select products for which to generate barcodes
  2. Choose barcode type (ID or SKU)
  3. Generate all barcodes at once
  4. Print in bulk if needed

Bulk Enable Products

Quick Outlet Setup:

  • Enable All Products for particular outlet by tapping Enable All Products button
  • One-click product assignment
  • Saves time for new outlets

Bulk Variation Stock

Set All Variations:

  • Click Set Variation Stock button to set stock for all variations
  • Apply same stock to all variations
  • Or set individual variation stock

Bulk Import Operations

Import Outlets:

  • Import outlets using sample_outlet.csv file
  • Bulk create multiple outlets
  • Include product assignments in import

Import Products:

  • Use WooCommerce product import
  • Include POS-specific data
  • Map POS fields correctly

Best Practices

Product Assignment

  1. Regular Review: Review product assignments monthly
  2. Seasonal Adjustments: Update products based on seasons
  3. Location-Specific: Assign products based on location demand
  4. Stock Levels: Maintain appropriate stock at each outlet

Inventory Management

  1. Master Stock: Use master stock for centralized control
  2. Regular Audits: Conduct physical inventory counts
  3. Alert Thresholds: Set appropriate low-stock thresholds
  4. Reorder Points: Establish reorder points for products

Barcode Management

  1. Consistent Format: Use either ID or SKU consistently
  2. Label Quality: Use quality barcode labels
  3. Scanner Testing: Test barcodes with your scanner
  4. Backup Codes: Keep digital backup of barcode assignments

Pricing Strategy

  1. Customer Segments: Define clear customer groups
  2. Discount Rules: Document discount policies
  3. Price Updates: Synchronize price changes across channels
  4. Coupon Management: Track coupon usage and effectiveness
