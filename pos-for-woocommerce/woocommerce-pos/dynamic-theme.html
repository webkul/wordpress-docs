WooCommerce POS - Dynamic Theme

Theme Mode Support

WooCommerce POS includes dynamic theme functionality that allows both admins and POS users to switch between light and dark modes based on their preferences.

Dark Theme Feature

System-Based Theme

The POS system works with the dark theme based on your system settings. The interface automatically adapts to match the operating system's theme preference.

Theme Configuration Options

Admin Control: The admin can change the light and dark mode settings from the admin panel.

Navigation:

Point of Sale → Settings → Appearance

POS User Control: The seller can switch light and dark modes in the POS panel via a button available on the panel.

Switching Theme in POS Panel

User-Level Theme Control

Steps to Change Theme:

  1. Open the POS panel
  2. Click on the theme switch in the right side corner
  3. Toggle between light and dark mode

The selected theme preference will be applied immediately to the POS interface.

Appearance Settings

POS Theme Mode

Admins can choose any theme mode according to their preferences by navigating to the Appearance section.

Configuration Path:

Point of Sale → Settings → Appearance

Available Theme Options

  • Light Mode
  • Dark Mode
  • System-based automatic switching

Theme Customization

POS Theme Colors

Color Schemes: From the dropdown menu, the admin can:

  • Select any pre-defined colour scheme
  • Set colours manually according to their preferences
  • Customize theme appearance to match brand identity

Theme Application

After saving the appearance settings according to admin preferences, the selected theme will reflect on the POS frontend interface.

Login Page Theme

The dark theme is also available on the POS login page, providing a consistent user experience from login through to active use of the system.
