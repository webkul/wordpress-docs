WooCommerce POS - Cash Drawer

Cash Drawer Overview

The WooCommerce POS system includes integrated cash drawer management functionality that helps POS agents track and manage cash transactions throughout the day.

Opening Cash Drawer

Initial Setup

When a POS agent logs into the system, they must enter the opening amount for the cash drawer.

Process:

  • A popup appears asking for "opening amount for the cash drawer"
  • POS agent enters the starting cash amount (e.g., $1000)
  • This becomes the required cash drawer amount for that particular day
  • Admin can enable/disable this popup from settings

Configuration Path:

Settings → POS Panel Configurations → Opening Amount Drawer Status

Cashier Management Features

The POS system provides three main functionalities under the "Cashier" tab:

1. Close Counter

Displays comprehensive drawer closing information:

Drawer Amount Details:

  • Opening amount (e.g., $1000.00)
  • Today's sales total
  • Expected amount in drawer (e.g., $1519.00)

Counted Drawer Amount:

  • Actual cash amount collected
  • Remarks section for notes

Closing Drawer Details:

  • Shows closing balance for the day
  • Displays difference between opening and closing amounts
  • Calculates discrepancies automatically

2. Today Cash

Shows current day's cash tracking:

  • Cash opening balance
  • Current cash balance
  • Current card balance
  • Order ID and timestamps
  • Payment mode for each transaction

Example Display:

  • Opening: $1000
  • Current Cash: $487.00
  • Current Card: $32.00

3. Sale History

  • View sale history for specific dates
  • Cash sale totals
  • Card sale totals
  • Combined total sales
  • Data visible only after drawer closure in "Close Counter" section

Admin Drawer Management

Drawer List & Reports

Admin can access comprehensive drawer management:

Navigation:

WooCommerce Point Of Sale → Drawer

Features:

  • View daily drawer details
  • Review closure details
  • Ensure proper completion of drawer process
  • Send email reports to users (bulk or individual)

Email Reports

Sending Options:

  • Bulk action: Send to multiple users
  • Individual report: Send to specific user
  • Confirmation message displayed after sending
  • Users receive formatted email with drawer details

Auto Drawer Close Feature

Cron Configuration:

Settings → POS Cron Configurations

Setup Options:

  • Set Cron Type: WP Cron or Server Cron
  • Enable/Disable auto drawer close
  • Set drawer closing time
  • Run commands in root directory

Drawer Close Email Functionality: Admin receives automatic email notifications when drawers are closed.
