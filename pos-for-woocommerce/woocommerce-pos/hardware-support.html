WooCommerce POS - Hardware Support

Supported Devices

Desktop & Laptop

  • Operating Systems: Windows, macOS, Linux
  • Minimum Screen Resolution: 1280x800 pixels
  • Recommended Resolution: 1280 screen resolution or above
  • Browser Compatibility: Modern web browsers (Chrome recommended)

Tablets

  • iPad: Fully supported
  • Android Tablets: Supported
  • Minimum Screen Size: 7.5 inches
  • Minimum Resolution: 1280x800 pixels

Mobile Devices

  • PWA (Progressive Web App): Mobile application support available
  • Development: Custom mobile app development available
  • Contact: support@webkul.com for mobile app development
  • Alternative: WooCommerce Flutter Native POS Mobile App purchasable

Desktop Application

  • Platform: PWA-based desktop app
  • Installation: Direct from Google Chrome browser
  • Click "+" icon in Chrome to install
  • Offline Capable: Works without internet connection
  • Sync: Auto-sync when online

Barcode Scanners

Barcode Support

  • Barcode Type: Code-128 barcodes
  • Scanning Methods:
    • USB Barcode Scanner
    • Bluetooth Barcode Scanner
    • Camera-based scanning (mobile devices)

Barcode Functionality

  • Product ID based barcodes
  • SKU based barcodes
  • Variable product barcodes (per variation)
  • Global Unique IDs support:
    • GTIN (Global Trade Item Number)
    • UPC (Universal Product Code)
    • EAN (European Article Number)
    • ISBN (International Standard Book Number)

Barcode Generation

  • Generate from Product ID
  • Generate from SKU
  • Bulk barcode generation
  • Print preview options (Portrait/Landscape)
  • Customizable barcode settings:
    • Background color
    • Width & Height
    • Text position & size
    • Display title enable/disable

Payment Hardware Integration

Supported Payment Terminals

  • Sunmi Terminal: Credit/Debit card payments
  • Square Payment Terminal: Via WooCommerce POS Square Payment Terminal plugin
  • Card Readers: Standard card reader support

Payment Methods

  • Cash drawer management
  • Card payment terminals
  • Split payment capability (Cash + Card)
  • Multiple payment gateway support

Receipt Printers

Thermal Printer Support

  • Epson TM-T88V Thermal Printer: Dedicated support
  • Industry-standard thermal printers compatible

Paper Size Options

  • A3 Printer
  • A4 Printer
  • A5 Printer
  • A6 Printer
  • Thermal Receipt Printer

Printer Configuration

POS Panel  Settings  Other Settings  Select Invoice Printer

Print Features

  • Auto-print enable/disable
  • Manual print option
  • Invoice logo customization
  • Custom invoice templates

Cash Drawer

Cash Drawer Integration

  • Opening amount popup
  • Electronic cash drawer support
  • Drawer open trigger on payment
  • Manual drawer control

Drawer Management Features

  • Opening amount entry
  • Closing amount calculation
  • Expected vs Actual amount comparison
  • Drawer close reports
  • Email notifications on drawer close

Configuration

Settings  POS Panel Configurations  Opening Amount Drawer Status
  • Enable/Disable drawer popup
  • Set drawer auto-close timing
  • Cron job setup for automatic operations

Display Hardware

Screen Requirements

  • Minimum: 1280x800 pixels
  • Recommended: 1920x1080 pixels or higher
  • Orientation: Landscape preferred
  • Touch Screen: Supported (optional)

Multi-Display Setup

  • Customer-facing display support (via hardware configuration)
  • Dual screen capability
  • Receipt printer display

Network & Connectivity

Internet Requirements

  • Online Mode: Active internet connection
  • Offline Mode: Works without internet
  • Sync: Auto-sync when connection restored
  • Node Server: Required for advanced features

Node Server Setup

  • Install Node.js on server
  • Configure domain/IP address
  • Set server port number
  • HTTPS/SSL support
  • Server timeout configuration

Server Requirements

Node Server Configuration:
- Domain/IP Address (without http/www)
- Server Port Number
- HTTPS Status (if SSL installed)
- API Request Timeout

Peripheral Device Support

Input Devices

  • Standard USB keyboard
  • Wireless keyboard
  • Keyboard shortcuts enabled
  • Barcode scanner (USB/Bluetooth)
  • Touch screen interface
  • Mouse/Trackpad

Keyboard Shortcuts

Enable from: POS Settings  Other Settings  Enable Keyboard Shortcuts

Available Shortcuts:

  • Header section actions
  • Menu navigation
  • Cart operations
  • Quick product add
  • Keyboard-only workflow

Mobile PWA App

Installation Process (Mobile)

  1. POS link ko mobile browser mein kholo
  2. Browser menu icon click karo
  3. "Add to Home Screen" select karo
  4. App icon home screen pe appear hoga
  5. Tap karke launch karo

PWA Features

  • Offline capability
  • Fast loading
  • Native app-like experience
  • Portrait view optimization
  • Low internet connection support
  • Auto-updates

Desktop PWA App

Installation Process (Desktop)

  1. Chrome browser mein POS login karo
  2. Address bar mein "+" icon click karo
  3. "Install" option select karo
  4. Desktop shortcut create hoga
  5. Standalone window mein launch hoga

Desktop App Benefits

  • Faster performance
  • Dedicated window
  • Offline functionality
  • Quick access from desktop
  • Full POS features
