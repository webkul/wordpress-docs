WooCommerce POS - WhatsApp Notification

WhatsApp Greetings Feature

WooCommerce POS includes WhatsApp integration functionality that allows store owners to send automated and manual messages to customers through WhatsApp.

Key Features

Automatic Order Receipts

  • Automatic sending of order receipts to customers via WhatsApp
  • Instant delivery after order completion
  • Includes order details and transaction information

Manual Messaging Option

  • Manual option to send order receipts available
  • Useful when automatic message feature is disabled
  • POS user controls when to send receipts

Broadcast Messaging

  • Broadcast WhatsApp messages to multiple customers simultaneously
  • Send promotional messages to customer groups
  • Efficient mass communication tool

Personalized Messages

  • Send personalized WhatsApp messages to any POS customer
  • One-to-one customer communication
  • Customized messaging capability

WhatsApp Greeting Templates

Template Management

Admins can manage and customize WhatsApp greeting templates through a dedicated interface.

Template Actions Available

  • Add new templates
  • Edit existing templates
  • Delete templates
  • Enable/disable templates
  • Sort templates
  • Search templates by status or name
  • Perform bulk actions

Creating Greeting Templates

Add New Template

Template Components:

WhatsApp Greeting Heading:

  • Enter the heading for your greeting message
  • Appears as the subject or title in the WhatsApp message

WhatsApp Greeting Content:

  • Add the main content of your greeting message
  • Choose between VisualText or Monospace formatting styles
  • Customize to suit design preferences

Greeting Status:

  • Choose whether template will be Enabled or Disabled for use

Save Template:

  • Click "Save Greeting Template" to save your new template

Template Customization

Store admins can edit existing templates by clicking on the edit option and customizing:

  • Template name
  • Message heading
  • Message content
  • Formatting style
  • Active status

API Configuration

Twilio Integration

To integrate Twilio with your WooCommerce store for WhatsApp messaging, configure the following fields. All credentials can be obtained from the Twilio Console.

Required Credentials:

WhatsApp Number:

  • Used for sending and receiving WhatsApp messages through Twilio
  • Essential for the integration

Twilio Account SID:

  • Unique identifier that links your Twilio account to the API
  • Necessary for authentication

Twilio Auth Token:

  • Secret key paired with the Account SID
  • Required to authenticate API requests
