WooCommerce POS - Managing Outlets & POS Users Guide

Creating POS Outlets

Overview

Create multiple store outlets for your online store to manage different physical locations effectively.

Accessing Outlets

Navigate to: WooCommerce Point of Sale > Outlets

Adding Outlets Manually

  1. Click Add New Outlet button
  2. Redirects to Add New Outlet page
  3. Fill in outlet details:
    • Outlet Name
    • Outlet Address
    • Contact Information
    • Operational Settings
  4. Save outlet configuration

Managing Existing Outlets

Bulk Actions

Select outlets to perform bulk operations:

  • Activate: Enable selected outlets
  • Deactivate: Disable selected outlets
  • Delete: Remove selected outlets

Individual Outlet Actions

  • Edit: Modify outlet settings
  • POS Login button: Redirects to POS agent login page for that outlet
  • Outlet Report: Click to view reports of each outlet
  • Manage Products: Assign and manage products for the outlet

Adding POS Agents/Users

User Management Overview

Admin can add, delete, and manage POS users by navigating to Point Of Sale > Users

Adding a New POS User

Step 1: Access User Creation

Navigate to: Point Of Sale > Users > Add New

Step 2: Enter User Details

Fill in the following information:

  • Username
  • Email Address
  • First Name
  • Last Name
  • Password
  • User Role (POS Agent/Manager)

Step 3: Outlet Assignment

  • Assign user to one or more outlets
  • Two or more users can be assigned to same outlet for proper management
  • Select primary outlet if applicable

Step 4: Additional Settings

  • Set user permissions
  • Configure access levels
  • Assign default customer if needed

Step 5: Save User

  • Review all settings
  • Click Save to create user

Default Customer Selection

Admin Configuration:

  • Admin can select any user to set default customer for POS from user's list
  • Default customer appears automatically at checkout
  • Useful for walk-in sales and quick transactions

Deleting POS Users

Individual Deletion:

  • Click delete icon next to user name
  • Confirm deletion

Bulk Deletion:

  • Select multiple users using checkboxes
  • Choose "Delete" from bulk actions dropdown
  • Confirm bulk deletion

Assigning Permissions

Store Agent Assignment

  • Admin Access: Admin can assign store agents to specific stores
  • Multi-Outlet Management: Outlet can have multiple agents assigned to it
  • Flexible Assignment: Agents can manage one or multiple outlets

Product Management Permissions

Product Visibility:

  • Admin can allow showing zero-price products in POS
  • Control which products appear in specific outlets

Product Assignment:

  • Admin can enable all products in POS outlet through dedicated button
  • Selective product assignment per outlet

Price Editing:

  • POS user can edit price of items in cart
  • Configure discount permissions
  • Customer-based discount applicable for few products

Order Management Permissions

Order Status Control:

  • Order status change permission available
  • By enabling this, POS manager can change order status
  • Track order progress through POS panel

Order Creation:

  • Create new orders from POS
  • Process returns and refunds
  • Hold orders for later completion

Inventory Control

Stock Management:

  • Assign POS stock to products for each outlet
  • Enable All Products for particular outlet by tapping Enable All Products button
  • Default product status for outlet can be set to enable or disable
  • Monitor stock levels in real-time

Report Access

Sales Reporting:

  • Sales agent can analyze store reports directly through POS end
  • View daily, weekly, and monthly reports
  • Track performance metrics

Report Types:

  • Sales reports
  • Product reports
  • Agent performance reports
  • Outlet comparison reports

Outlet-Specific Settings

Product Assignment

Manual Assignment:

  • Assign products to outlet by enabling products for outlet
  • Select specific products for each location

Bulk Assignment:

  • Enable All Products for particular outlet by tapping Enable All Products button
  • Quick setup for new outlets

Stock Assignment:

  • Assign POS stock to products for each outlet
  • Admin edits outlet and manages products to add POS stock to product accordingly
  • Set minimum and maximum stock levels

Product Status Configuration

  • Default Product Status: Default product status for outlet can be set enable or disable
  • Control product visibility per outlet
  • Manage seasonal or location-specific products

Payment Methods

Outlet-Specific Payments:

  • Added payment method must be selected in respective outlet to be visible at POS panel
  • Configure different payment methods for different outlets
  • Enable/disable payment options per location

Outlet Reports

Accessing Reports:

  • Admin can click on Outlet Report to view reports of each outlet
  • Compare performance across locations
  • Identify trends and opportunities

Report Features:

  • Sales summaries
  • Product performance
  • Agent activity
  • Payment method breakdown

Managing Agent Roles

Store Agent Responsibilities

Store Management:

  • Admin can assign store agent to manage physical stores
  • Agents handle day-to-day operations
  • Oversee staff and inventory

User Credentials:

  • Default user credentials added in Setup Wizard
  • Admins provide login details to agents
  • Secure credential management

Agent Capabilities

Profile Management:

  • POS user can change their profile image
  • Update personal information
  • Manage account settings

Sales Management:

  • Better sales management by POS agent by monitoring:
    • Sales history
    • Holding sales
    • Offline sales from POS panel
  • Process transactions efficiently

Report Analysis:

  • Sales agent can analyze store reports directly through POS end
  • Make data-driven decisions
  • Track personal performance

Product Management by Agents

New Product Sales:

  • POS agents can sell new product not available on website
  • Add products on-the-fly
  • Handle special orders

Inventory Control:

  • Monitor stock levels
  • Request restocking
  • Report discrepancies

Customer Management

Customer Search:

  • POS user can search for customers by name or email
  • Quick customer lookup
  • Access customer history

Discount Application:

  • Customer-based discount applicable for few products
  • Apply loyalty discounts
  • Handle special pricing

Login & Access Control

POS Login Process

Agent Login:

  1. POS agent enters login details provided by admin to access front end
  2. Default user credentials added in Setup Wizard
  3. Secure authentication process

Login Screen Features:

  • Username/Email field
  • Password field
  • Outlet selection (if multiple outlets assigned)
  • Remember me option

Opening Cash Drawer

Cash Drawer Management:

  • After logging in, POS agent enters opening amount for cash drawer
  • Popup can be enabled/disabled by admin
  • Record starting cash for shift reconciliation

Navigation Menu

Left Navigation Drawer displays menu options:

  • Home: Main dashboard and quick actions
  • Customer: Customer management and search
  • Cashier: Point of sale interface
  • Orders: Order management and history
  • Report: Sales and performance reports
  • Group Message: Broadcast messaging feature
  • Settings: POS configuration options
  • Reset: Data synchronization options
  • Logout: Secure logout

Access Levels

Admin Access:

  • Full system configuration
  • All outlets and users
  • Global settings management
  • Report access across all outlets

Agent Access:

  • Assigned outlet(s) only
  • Limited configuration options
  • Sales and customer management
  • Outlet-specific reports

POS System Features:

  • Different access levels for admin vs agents
  • Role-based permissions
  • Secure data segregation

Multi-Site Compatibility

Multi-Site Features:

  • POS is now multisite compatible
  • Manage multiple WordPress installations
  • Centralized user management
  • Shared customer database (optional)

Reset Functionality

Reset Options:

Reset button in WooCommerce Point Of Sales gives power to POS user to refresh POS front end:

  • Resync All: Reset all POS data
  • Resync Products: Reset products only
  • Resync Orders: Reset orders only
  • Resync Customers: Reset customers only

When to Use Reset:

  • After major updates
  • Synchronization issues
  • Data discrepancies
  • Fresh start needed

Best Practices

Outlet Management

  1. Naming Conventions: Use clear, consistent naming for outlets
  2. Regular Updates: Keep outlet information current
  3. Product Assignment: Review assigned products regularly
  4. Stock Monitoring: Check stock levels frequently

User Management

  1. Access Control: Grant minimum necessary permissions
  2. Regular Audits: Review user access periodically
  3. Password Policy: Enforce strong password requirements
  4. Training: Ensure agents are properly trained

Security

  1. Credential Protection: Keep login credentials secure
  2. Role Segregation: Use appropriate role assignments
  3. Activity Monitoring: Review agent activity logs
  4. Regular Updates: Keep POS system updated

Performance Optimization

  1. Data Cleanup: Remove old or inactive data regularly
  2. Cache Management: Clear cache when needed
  3. Synchronization: Run regular sync operations
  4. Backup: Maintain regular backups

Troubleshooting

Common Outlet Issues

Outlet Not Showing:

  • Verify outlet is activated
  • Check product assignments
  • Ensure proper configuration

Sync Problems:

  • Use reset functionality
  • Check internet connection
  • Verify cron job settings

User Access Issues

Login Failed:

  • Verify credentials are correct
  • Check user is activated
  • Ensure outlet assignment is valid

Permission Denied:

  • Review user role and permissions
  • Check outlet-specific settings
  • Contact administrator

Product Assignment Issues

Products Not Visible:

  • Check product is enabled for outlet
  • Verify stock levels
  • Review product status settings

Stock Discrepancies:

  • Run product resync
  • Check manual stock adjustments
  • Review order history
