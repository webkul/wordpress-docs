WooCommerce Power BI Connector Documentation

Sync WooCommerce data to Microsoft Power BI. Configure workspaces, embed reports, schedule syncs, and set up alerts - all from your WordPress admin.

Quick StartInstallationBuy Now

Real-Time Data Sync

Sync 16 WooCommerce data types — orders, products, customers, refunds, coupons, and more — to Power BI via live, scheduled, or manual modes.

Embedded Reports

Embed Power BI reports and dashboards inside WordPress pages using shortcodes or the admin panel. Supports iframe and JS embed methods.

7-Step Setup Wizard

Register an Azure AD app, connect to Power BI, select a workspace, and run your first sync — the wizard walks you through each step.

Alerts & Notifications

Configure threshold-based alerts for revenue, order count, AOV, refund rate, and stock levels. Notifications are sent via email.

Visual Query Builder

Build custom Power BI queries without writing code. Select columns, apply filters, preview results, and export directly from the admin.

Access Control & Security

Azure AD OAuth2 authentication, encrypted credentials, 6 role-based capabilities, and a full audit log of every sync and user action.
