Coupons Import

Bulk-create discount codes for your store.

Step 2 maps your source columns to coupon fields: code, discount_type, coupon_amount, usage limits, and email restrictions.

When to Use

  • Seasonal promos (500 unique codes for a Black Friday campaign)
  • VIP customer codes
  • Bulk email marketing campaigns
  • Restoring coupons from a backup

Match Keys

  • code (default) — The coupon code
  • id — WordPress post ID

Column Reference

Basic Fields

ColumnRequiredExample
idNo42 (for updates)
codeYesSUMMER25
descriptionNo25% off summer collection
discount_typeNo (default fixed_cart)percent, fixed_cart, fixed_product
coupon_amountYes25 (percent or dollars)
amountAlternative to coupon_amount25
date_expiresNo2026-12-31 23:59:59

Discount Types

TypeWhat It Does
percentPercentage off total
fixed_cartFixed amount off cart subtotal
fixed_productFixed amount off each product

Usage Limits

ColumnExample
usage_limit100 (total uses)
usage_limit_per_user1 (per customer)
limit_usage_to_x_items5 (only applies to first 5 items)
usage_count42 (current usage, for migrations)

Minimum and Maximum

ColumnExample
minimum_amount50.00
maximum_amount500.00

Free Shipping

ColumnExample
free_shipping1 (yes) or 0 (no)

Exclusions

ColumnExample
exclude_sale_items1 or 0
individual_use1 (cannot combine with others) or 0

Product Restrictions

ColumnExample
product_ids123,124,125 (comma-separated product IDs)
excluded_product_ids200,201
product_categories5,10 (category IDs)
excluded_product_categories99

Email Restrictions

ColumnExample
customer_emailcustomer@example.com,*@company.com
email_restrictionsAlternative name for same field

Use *@domain.com for whole-domain restrictions.

Complete Example

code,description,discount_type,coupon_amount,minimum_amount,usage_limit,date_expires,free_shipping,individual_use,product_categories
SUMMER25,Summer sale 25% off,percent,25,50.00,1000,2026-09-30 23:59:59,0,1,5
SHIP10,$10 off shipping,fixed_cart,10,75.00,500,2026-12-31 23:59:59,1,0,
VIPFREE,VIP free shipping,percent,0,0,,2026-12-31 23:59:59,1,0,

Use Cases

Bulk Unique Codes for a Campaign

Create 100 unique codes:

code,discount_type,coupon_amount,usage_limit_per_user
SAVE001,percent,10,1
SAVE002,percent,10,1
SAVE003,percent,10,1

VIP Codes Tied to Specific Emails

code,discount_type,coupon_amount,customer_email
VIP-JOHN-2026,percent,20,john@example.com
VIP-JANE-2026,percent,20,jane@example.com

Expire-Soon Flash Sale

code,discount_type,coupon_amount,date_expires
FLASH50,percent,50,2026-04-15 23:59:59

Troubleshooting

ProblemFix
Coupon created but not workingCheck start/end dates, usage limits, and minimum spend
Wrong discount typeValues are case-sensitive: percent not Percent
Expired on creationCheck date_expires is in the future
Product restriction ignoredUse product_ids (numeric IDs)
Category restriction not workingproduct_categories must be IDs, not names

The results screen shows how many codes were created, updated, or skipped — handy proof when you push a thousand unique Black Friday codes at once.
