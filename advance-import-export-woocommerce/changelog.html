Changelog

All notable changes to the plugin.

1.0.0 — Initial Release

Features

Import

  • 14 entity types (products, orders, customers, categories, tags, attributes, coupons, reviews, posts, pages, users, subscriptions, bookings, memberships)
  • 11 import sources (file, server path, URL, FTP, SFTP, Google Sheets, Google Drive, Dropbox, Amazon S3, REST API, WC REST API)
  • 5 import modes (Create Only, Update Only, Create + Update, Delete, Replace)
  • 6 file formats (auto-detect, CSV, XML, JSON, XLSX, ODS)
  • ZIP and TAR archive auto-extraction
  • Quick Import Wizard (5 steps)
  • Drag-and-drop column mapping with auto-match
  • Value transformations (find/replace, price rules, value mapping)
  • Preview / dry run before committing
  • Migration presets for Shopify, Magento 1/2, BigCommerce, PrestaShop, OpenCart, Amazon, eBay
  • Delta import (SHA-256 hash-based skip)

Export

  • 15 entity types
  • 3 engines (Plugin pipeline, WordPress native, WooCommerce native)
  • 3 formats (CSV, JSON, XML) + WXR for WP native
  • 12 export destinations (download, server, FTP, SFTP, Google Sheets, Google Drive, Dropbox, OneDrive, Amazon S3, REST API, WC REST API, email)
  • Product feed templates (Google Shopping, Facebook, Amazon, eBay, Pinterest, Bing)
  • Streaming export for large datasets

Automation

  • Saved jobs with 6-tab editor
  • Cron schedules (every minute to monthly, or custom cron)
  • Event triggers (product save, new order, stock change, custom hook)
  • Job chaining
  • REST API (16 endpoints)
  • WP-CLI (8 command groups)

Safety

  • One-click rollback with selective undo
  • 23 health checks with auto-fix
  • Post-import verification
  • Run history with logs per run

Third-Party Compatibility

  • Yoast SEO (10 meta fields)
  • RankMath SEO (9 meta fields + schema JSON)
  • All in One SEO (6 meta fields)
  • Advanced Custom Fields (ACF)
  • Meta Box
  • WPML
  • Polylang
  • Dokan
  • WCFM Marketplace
  • WC Vendors
  • WooCommerce Brands
  • WooCommerce Product Bundles
  • WooCommerce Composite Products
  • WooCommerce Product Add-ons

Admin UI

  • Dashboard with stat cards, recent activity, system info
  • System info card with OS-aware fix instructions
  • Settings across 7 tabs
  • History page with filters
  • Health Check page with 23 checks
  • Job Editor with 6 tabs

Performance

  • 10,000 products imported in seconds
  • 1 million products imported in a few minutes
  • Parallel processing support
  • Streaming for large files
  • Delta import for repeated syncs

Known Issues

None critical at launch. See Troubleshooting for edge cases.
