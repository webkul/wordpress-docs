Product Listing – WooCommerce eBay Connector

This guide explains how to import products from eBay to WooCommerce.

Import Products from eBay

The admin can import products from eBay to the WooCommerce store.

Navigate to WC eBay Connector → Manage Products to see the import options:

  • Import Products From eBay – Import all active eBay products
  • Import Products Date-wise – Import multiple products according to specific dates

Steps to Import

  1. Click Import Products from eBay to start the import.
  2. This will create temporary product data and store it in the WooCommerce store.
  3. A background process for image import also runs during product import.
  4. You can start and stop the product image import process as required.

Note: Only products that are currently listed as active on eBay will be imported.

Category Assignment During Import

  • If category mapping is not done for the eBay product category, imported products will be assigned to the Default Category as per configuration.
  • If category mapping is done, imported products will be assigned to the mapped category.
  • If Auto Mapping is enabled, products imported from eBay will be assigned to the eBay category already imported in WooCommerce.

Refresh Products

The Refresh Product button lets you update your product information from eBay automatically.

When you click the button, a popup will appear notifying you that this is a time-consuming process. It advises you to use the Refresh eBay Products option only if:

  • Real-time sync is off, or
  • Products are out of sync

Once you click Yes, your product listings will be updated with the latest details from eBay.

Synced Product Data

The following product data is synced between WooCommerce and eBay:

  • SKU
  • Product Name / Title
  • Description
  • Stock / Inventory
  • Images
  • Categories
  • Price
  • eBay Conditions & Specifications
  • Attributes
