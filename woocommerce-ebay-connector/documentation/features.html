Features

The WooCommerce eBay Connector module integrates your WooCommerce store with the eBay marketplace, providing a robust two-way synchronization for products, orders, and inventory management.

Core Features of the Connector

  • Two-Way Product Synchronization: Seamlessly import eBay products to WooCommerce and export WooCommerce products to eBay.
  • Support for Product Types: Synchronize both Simple and Variable products across platforms.
  • Category Mapping: Map eBay categories (including custom shop categories) with WooCommerce store categories for organized listings.
  • Order Management: Automatically import eBay orders into the WooCommerce dashboard for centralized processing.
  • Real-Time Synchronization: The admin can enable real-time updates for product creation, revision, and deletion events.
  • Automatic Inventory Management: Ensures stock levels are always consistent between the WooCommerce store and the eBay marketplace.
  • Multi-Account Support: The admin can configure and manage multiple eBay accounts from a single WooCommerce backend.
  • Price Rule Management: Create rules to automatically increase or decrease product prices during import or export.
  • Global Marketplace Support: Compatible with various global eBay sites including US, UK, AU, India, and more.
  • Automated Sync Logs: Maintain separate logs for all synchronization events to monitor and debug activities.
  • Sandbox Testing: Includes a Sandbox mode for testing configurations before going live.
  • Detailed Product Sync: Synchronizes SKU, Title, Description, Stock, Images, and Attributes along with the products.
  • Return & Shipping Policy: The admin can configure default return and shipping details for eBay listings.

Supported eBay Marketplaces

The module supports a wide range of global eBay marketplaces, including:

  • eBay Motors
  • eBay United States & Canada (English/French)
  • eBay UK & Ireland
  • eBay Australia & New Zealand
  • European Sites: France, Germany, Italy, Spain, Switzerland, Netherlands, Belgium, Austria, Poland.
  • Asian Sites: India, Hong Kong, Malaysia, Philippines, Singapore.

eBay Motors Integration

One of the standout features of this module is its eBay Motors integration, built specifically for automotive sellers.

With the WooCommerce eBay Item Compatibility module, you can:

  • Import vehicle compatibility listings
  • Ensure accurate product-to-vehicle matching
  • Improve product visibility and buyer confidence

Real-Time Synchronization Features

The eBay Connector allows the admin to automate updates based on specific marketplace events:

Automatic Creation

  • When a new product is added to the eBay store, it is automatically created in WooCommerce.
  • Synced details include category, title, SKU, description, price, stock, images, etc.

Automatic Revision

  • Any edits made to products on eBay (price, stock, details) are instantly reflected in the WooCommerce store.
  • This ensures product data is always accurate without manual intervention.

Automatic Deletion

  • If a product listing ends or is deleted on eBay, the system can automatically update the linked product in WooCommerce.

Automatic Inventory Management

The module streamlines inventory control by reducing manual work and preventing overselling:

  • Sales Synchronization: When a product is sold on eBay, WooCommerce automatically adjusts the corresponding stock level.
  • Out of Stock Management: Products marked as out of stock on eBay are instantly updated in WooCommerce.
  • Bidirectional Flow: This automatic inventory sync works both ways based on the real-time settings, ensuring WooCommerce changes also reflect on eBay.
