Features – WooCommerce eBay Connector

The WooCommerce eBay Connector plugin integrates your WooCommerce store with eBay, providing two-way synchronization for products, orders, and inventory.

Core Features

  • Import simple and variable eBay products in the WooCommerce store.
  • Import custom-created shop categories from eBay to the WooCommerce store.
  • The connector has Automatic Inventory Management functionality.
  • Import eBay orders in the WooCommerce store.
  • Export simple and variable products from the WooCommerce store to eBay.
  • Map custom-created shop categories on eBay with the WooCommerce store category.
  • Admin will update imported product details.
  • Admin will configure the return policy details for exported products.
  • The product's SKU, name, description, stock, images, and categories export and import along with the product.
  • Supports product synchronization with eBay conditions and specifications.
  • Admin will configure multiple eBay accounts.
  • Sandbox mode is available for testing and development.
  • Configure default product payment and shipping details.
  • Admin will set the events in the Realtime setting for the auto-update of the product.
  • Admin will select the process for the auto-update.
  • Admin will also configure the International Shipping Service.
  • Admin will manage the Real-Time Sync and update product information as per the selected event.
  • The count of the products added at the backend can be checked.
  • Separate logs for real-time sync events.

Real-Time Synchronization Features

The connector supports the following Real-Time Synchronization features:

Automatic Creation

When a product is created in the eBay store, its updates will reflect in the WooCommerce store. The synced details include the category, description, title, SKU, price, stock, images, and attributes.

Automatic Update

When a product is edited in the eBay store, the update reflects in the WooCommerce store. The synced details include the category, description, title, SKU, price, stock, images, and attributes.

Automatic Deletion

When an item is deleted from the eBay store, the change reflects in the WooCommerce store.

Automatic Inventory Management

The WooCommerce eBay Connector features Automatic Inventory Management, streamlining inventory control and reducing manual work.

  • When a product is sold or updated on eBay, WooCommerce automatically adjusts its stock.
  • If a product goes out of stock on eBay, it is also marked out of stock in WooCommerce.
  • This automatic inventory sync works both ways, based on the Real-Time Sync settings, ensuring WooCommerce changes also reflect on eBay.

Compatibility

  • Compatible with WooCommerce Cart and Checkout blocks
  • Compatible with High-Performance Order Storage (HPOS)
  • Supports multiple stores simultaneously

Note: The Connector supports pictures in the following file formats: JPEG, BMP, GIF, TIF, PNG.
