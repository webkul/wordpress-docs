Introduction – WooCommerce eBay Connector

The WooCommerce eBay Connector module integrates your WooCommerce store with eBay, allowing the admin to manage both platforms efficiently from a single dashboard.

eBay is a leading global e-commerce marketplace that facilitates consumer-to-consumer and business-to-consumer sales. Using this connector, the admin can buy and sell products globally and synchronize store data across multiple regions.

Module Overview

The eBay Connector provides a robust two-way synchronization system, enabling the admin to sync products, categories, and orders between the WooCommerce site and the eBay marketplace.

Key Capabilities for the Admin

  • Import Synchronized Data: Fetch product information, categories, and orders directly from eBay to WooCommerce.
  • Custom Category Import: Import customized shop categories from the eBay store to the WooCommerce dashboard.
  • Product Export: Export simple and variable products from WooCommerce to eBay listings.
  • Direct Listing: Add items from the WooCommerce site directly to the connected eBay seller account.
  • Policy Management: Configure eBay conditions, specifications, refund policies, and shipping details for all exported products.

Technical Compatibility

Before installing the module, the admin must ensure the following system requirements are met:

RequirementMinimum / Recommended Version
WordPress6.7 or higher
WooCommerce10.0 or higher
PHP7.4+ (Standard and Recommended)
SSL CertificateEnabled and Active

Important Module Features

  • Modern Block Support: Fully compatible with WooCommerce Cart and Checkout blocks.
  • Performance Optimization: Supports High-Performance Order Storage (HPOS) for faster processing.
  • Multi-Store Integration: Manage multiple eBay accounts and stores simultaneously from a single module.
  • Supported Image Formats: Compatible with JPEG, BMP, GIF, TIF, and PNG (refer to the eBay Developer Guide for details).
