FAQs – WooCommerce eBay Connector

Find answers to the most commonly asked questions about the WooCommerce eBay Connector plugin.

General Questions

What is WooCommerce eBay Connector?

WooCommerce eBay Connector is a plugin that integrates your WooCommerce store with eBay marketplace. It allows you to list products on eBay, import orders, and synchronize inventory between both platforms.

Do I need an eBay Developer Account?

Yes, you need an eBay Developer Account to obtain API credentials (App ID, Dev ID, Cert ID) required to connect your store with eBay. Sign up for free at eBay Developer Program.

Which eBay marketplaces are supported?

The module supports a wide range of global eBay marketplaces, including:

  • eBay Motors: Support for vehicle compatibility listings is available through the WooCommerce eBay Item Compatibility module.
  • eBay United States & Canada (English/French)
  • eBay UK & Ireland
  • eBay Australia & New Zealand
  • European Sites: France, Germany, Italy, Spain, Switzerland, Netherlands, Belgium, Austria, Poland.
  • Asian Sites: India, Hong Kong, Malaysia, Philippines, Singapore.

Can I use the module with existing eBay listings?

Yes, the module can work with existing eBay listings. The admin can link existing listings to WooCommerce products for synchronization.

What are the system requirements?

Before installation, make sure:

WordPress6.7 or higher
WooCommerce10.0 or higher
PHP7.4+ (recommended)
SSLEnabled (recommended)

How do I install the plugin?

Refer to our Installation Guide for step-by-step instructions.

Why am I getting an API connection error?

Common causes:

  1. Incorrect API Credentials - Double-check your App ID, Dev ID, and Cert ID
  2. Server IP Not Whitelisted - Add your server IP to eBay Developer settings
  3. Firewall Blocking - Ensure your server allows outbound connections
  4. Expired Token - Re-authorize your eBay account

How do I get eBay API credentials?

  1. Sign up at eBay Developer Program
  2. Create a new application
  3. Generate API keys for Sandbox or Production
  4. Copy the credentials to plugin settings

Product Listing

How do I list products on eBay?

  1. Go to WooCommerce → eBay Connector → Export Products
  2. Select products to list
  3. Apply a profile with listing settings
  4. Review and submit to eBay

Can I list variable products?

Yes, the plugin fully supports variable products. All variations will be listed on eBay with their respective attributes, prices, and stock levels.

Why are my product images not showing on eBay?

Common reasons:

  • Image Size - Images must be at least 500px
  • Image Format - Use JPG, PNG, or GIF
  • Image URL - Images must be publicly accessible
  • Image Count - eBay allows up to 12 images

Can I set different prices for eBay?

Yes, configure price adjustments in profile settings:

  • Percentage Markup - Add a percentage (e.g., +10%)
  • Fixed Markup - Add a fixed amount (e.g., +$5)
  • Custom Price - Set specific eBay prices per product

Order Management

How are eBay orders imported?

eBay orders are automatically imported to WooCommerce based on your sync settings. The import process:

  1. Checks for new eBay orders
  2. Creates WooCommerce orders with customer data
  3. Maps order items to products
  4. Sets appropriate order status

Can I manage eBay orders from WooCommerce?

Yes, all eBay orders appear in WooCommerce Orders. You can:

  • Update order status
  • Add tracking numbers
  • Generate shipping labels
  • Process refunds

How do order statuses sync?

eBay StatusWooCommerce Status
ActiveProcessing
CompletedCompleted
CancelledCancelled
RefundedRefunded

What happens when an order is cancelled on eBay?

The corresponding WooCommerce order status is updated to "Cancelled" automatically.

Inventory & Synchronization

How does inventory synchronization work?

The plugin keeps inventory synchronized:

  • WooCommerce → eBay - Stock changes update eBay listing
  • eBay → WooCommerce - eBay sales reduce WooCommerce stock
  • Bidirectional - Both platforms stay in sync

What happens when a product goes out of stock?

Based on your settings, the plugin can:

  • End the listing - Remove from eBay
  • Set quantity to 0 - Show as out of stock
  • Do nothing - Leave listing as-is

How often does synchronization occur?

Sync TypeDefault Frequency
OrdersEvery 5 minutes
InventoryEvery 15 minutes
PricesEvery 15 minutes

Can I sync only specific products?

Yes, you can:

  • Select specific products for eBay listing
  • Enable/disable sync per product
  • Create profiles for different product groups

Multi-Account & Multi-Store

Can I connect multiple eBay accounts?

Yes, the plugin supports multiple eBay accounts:

  • Connect unlimited eBay seller accounts
  • Assign products to specific accounts
  • Manage orders from all accounts centrally

Can I use one eBay account with multiple WooCommerce stores?

No, each eBay account can only be connected to one WooCommerce store at a time.

Troubleshooting

Why are my products not listing on eBay?

Common causes:

  1. Missing Required Fields - Ensure products have title, price, description
  2. Invalid Category - Check eBay category mapping
  3. API Limits - You may have reached eBay API rate limits
  4. Account Issues - Verify your eBay seller account is in good standing

How do I enable debug mode?

  1. Go to WooCommerce → eBay Connector → Settings → Advanced
  2. Enable Debug Mode
  3. View logs in Tools → Logs

Where can I find error logs?

Error logs are available at: WooCommerce → eBay Connector → Tools → Logs

How do I fix "Token Expired" error?

  1. Go to Settings → eBay Account
  2. Click Disconnect
  3. Click Connect with eBay
  4. Re-authorize your account

Billing & Licensing

How is the plugin licensed?

The plugin is licensed per domain. Each license includes:

  • 1 year of updates
  • 1 year of support
  • Access to all features

Can I use the plugin on multiple sites?

Each license is valid for one production site. For multiple sites, additional licenses are required.

What happens when my license expires?

After expiration:

  • The plugin continues to work
  • You won't receive updates
  • Support will be limited
