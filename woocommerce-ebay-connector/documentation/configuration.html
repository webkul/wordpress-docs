Configuration – WooCommerce eBay Connector

After successful installation and account setup, configure the plugin settings by navigating to WC eBay Connector → eBay Settings.

General Settings

Product Import Settings

SettingDescription
Default CategoryIf an eBay category isn't mapped in WooCommerce, products are placed in this default category
eBay Imported Order StatusSet default order status for orders imported from eBay
Default Product QuantitySet the default product quantity for a product exported to eBay if the product quantity is zero in the store
Email Address for ReportEnter the email address to receive reports of every update sync
Pin code of Item LocationEnter the pincode of the item location
Global ShippingSelect Yes/No for global shipping depending on business need
Default Condition for Exported ProductIf no product condition is selected, this default condition will be applied to exported products
Create Active Listing OnlySelect Yes to create active listings only, else select No
Create Fixed Price ListingSelect All Products or Fixed Price Products
Product Image ImportSelect In the Background (faster for large imports) or During Product Import
Product Import Only from Global SiteIf enabled, only products belonging to the eBay global site chosen in settings will be imported

Note: For large numbers of product imports, it is advisable to keep the Product Image Import setting as In the Background to import products at a faster pace.

Product Export Settings

SettingDescription
Default Product QuantitySet the default quantity to export to eBay if the product quantity on store is zero
Pin code of Item LocationEnter the pin code of the item location
Global ShippingSelect Yes or No as per business requirement
Default ConditionSelect the default condition of the product listing when exported to eBay (New, Open Box, Manufacturer Refurbished, Seller Refurbished, Used, etc.)

Auto Mapping

Enable Auto Mapping: Enable or disable the category auto-mapping feature. Auto Order Import: Select the condition for auto order import. It executes every 10 minutes and retrieves orders placed within the last 15 minutes.

Note: All products must be imported for real-time order import.

Real-Time Sync Settings

Navigate to WC eBay Connector → eBay Settings → Real-Time Sync to manage synchronization settings.

The admin can enable real-time sync to update the WooCommerce store the same as the eBay store. Product information can be updated based on:

  • Item Created
  • Item Revised
  • Item Sold-out
  • Item Closed

Note: For real-time settings, eBay Connector does not work for sandbox credentials.

eBay to WooCommerce Real-Time Sync

Enable the status for eBay to WooCommerce real-time sync.

Select eBay Update Events

EventDescription
CreateWhen a product is added on eBay, WooCommerce syncs its category, description, title, SKU, price, stock, images, and attributes. Auction-type products import as Simple type products.
RevisedWooCommerce automatically updates products edited on eBay with all details including category, title, SKU, price, stock, images, and attributes.
Sold-outWhen an item goes out of stock on eBay, WooCommerce updates its stock accordingly.
ClosedDeleting an item on eBay automatically reflects in WooCommerce.
TransactionWhen an item is purchased from the eBay store, the updates get reflected in the WooCommerce store.

WooCommerce to eBay Real-Time Sync

Enable real-time sync between WooCommerce Store and eBay. Any updates at the WooCommerce store end will reflect at the eBay end instantly.

Synced Details (WooCommerce → eBay)

  • Images
  • Description
  • SKU
  • Title
  • Category
  • Condition ID
  • Price
  • Stock
  • Return Policy

Note: For Item Sold and Refund, only stock updates will be applied.

Automatic Inventory Management

The WooCommerce eBay Connector features Automatic Inventory Management, streamlining inventory control and reducing manual work.

  • When a product is sold or updated on eBay, WooCommerce automatically adjusts its stock.
  • If a product goes out of stock on eBay, it is also marked out of stock in WooCommerce.
  • This automatic inventory sync works both ways, based on the Real-Time Sync settings, ensuring WooCommerce changes also reflect on eBay.

Price Rule Fields

The store admin can add price rules for the WooCommerce products from Add Price Rule tab.

FieldDescription
Product Price FromSet the starting price to apply the rule
Product Price ToSet the last price for the applied rule
PriceSet the price for the price rule
Operation TypeSelect Fixed or Percentage
OperationSet the price rule operation as Increase or Decrease
Price Rule StatusEnable or disable the price rule
Price Rule Applicable ForSelect Import, Export, or None

Click Save to create the price rule.

Price Rule List

The created price rules for products will display in a list after saving.

Bulk Actions

A button to Enable, Disable, or Delete a created price rule is provided for mass action.
