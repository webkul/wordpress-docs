Helpful Details to Include

  1. Error Messages - Copy the exact error message
  2. Steps to Reproduce - List the steps that cause the issue
  3. Screenshots - Capture any visual errors
  4. Recent Changes - Note any recent updates or changes
  5. Debug Logs - Enable debug mode and attach logs

How to Enable Debug Mode

// Add to wp-config.php
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
define('WP_DEBUG_DISPLAY', false);

Debug logs will be saved to /wp-content/debug.log
