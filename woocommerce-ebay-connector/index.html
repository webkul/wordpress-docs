WooCommerce eBay Connector Documentation

Integrate your WooCommerce store with eBay using WooCommerce eBay Connector. Easily import and export products, sync orders in real time, and efficiently manage inventory across both platforms.

👉 Get StartedView DemoBuy Now

Two-Way Product Sync

Import simple and variable eBay products to WooCommerce and export WooCommerce products to eBay using WooCommerce eBay Connector.

Order Import

Import eBay orders directly into your WooCommerce store automatically as outlined in WooCommerce eBay Connector Documentation.

Automatic Inventory Management

Automatically adjust stock levels in both WooCommerce and eBay when products are sold or updated as per the documentation for eBay connector.

eBay Account Setup

Test your integration safely using eBay Sandbox mode before going live.

Category Mapping

Map WooCommerce categories to eBay categories and import eBay shop categories in WooCommerce eBay Connector.

Multi-Account Support

Configure and manage multiple eBay accounts from a single WooCommerce store through the WooCommerce eBay Connector Documentation.

Real-Time Sync

Sync product creation, updates, deletions, and sales events in real time as per the WooCommerce eBay Connector.

Shipping Configuration

Configure domestic and international shipping services, dispatch time, and costs.

Price Rules

Add price rules for import/export with fixed or percentage increase/decrease operations as explained in the documentation for eBay connector.
